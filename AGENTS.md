# API CLI

A CLI for the Autodesk Platform Services (APS)

## Use the CLI

`aps --help`

See skill.md

## How to Work

After all changes call the CLI without building it using `aps <COMMAND>`. The env is setup for it to run without the build step because this version of node supports .ts.
